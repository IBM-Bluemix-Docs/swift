# 新增 Swift SDK 至應用程式
{: #sdk}

若要新增 {{site.data.keyword.cloud}} SDK 至應用程式，請選擇您要使用的 SDK，然後配置您的相依關係管理程式，以將 SDK 拉進您的應用程式。

{: #client_sdk}
